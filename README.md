# 📖 منصة SujetStore التعليمية

> **منصة تعليمية للمنهج الجزائري**  
> _Algerian Educational Curriculum Platform_

منصة **SujetStore** هي منصة تعليمية سريعة وحديثة، مبنية كبديل مفتوح وخفيف لمواقع الفروض والاختبارات الجزائرية (مثل DzExams). المنصة مخصصة لعرض الفروض، والاختبارات، والدروس، والملخصات لجميع المراحل الدراسية (ابتدائي، متوسط، ثانوي) بواجهة مستخدم عصرية وسهلة الاستخدام.

## 🚀 التشغيل السريع (Quick Start)

```bash
# 1. تثبيت التبعيات | Install dependencies
bun install

# 2. إنشاء قاعدة البيانات وإضافة البيانات الأولية | Seed Database
# سيقوم هذا السكربت بإنشاء content.db وإضافة نموذج أساسي للمدارس والمواد والفروض
bun run scripts/seed-education.ts

# 3. تشغيل خادم التطوير | Start Dev Server
bun run dev --open
```

## 🛠️ التقنيات المستخدمة (Tech Stack)

تم بناء المشروع لتوفير أداء فائق وسرعة لا تضاهى في عرض المحتوى:

| المكون        | التقنية                  | السبب                                                  |
| ------------- | ------------------------ | ------------------------------------------------------ |
| **Runtime**   | **Bun**                  | أداء أسرع بأضعاف من Node.js التقليدي                   |
| **Framework** | **SvelteKit (Svelte 5)** | أداء عالي وسهولة تطوير (Full-Stack)                    |
| **Database**  | **SQLite**               | قاعدة بيانات مدمجة، سريعة جداً، ومثالية لقراءة المحتوى |
| **ORM**       | **Drizzle**              | Type-safe وخفيفة جداً                                  |
| **CSS**       | **TailwindCSS v4**       | تصميم متجاوب، أنيق، ومباشر                             |
| **UI**        | **Shadcn-Svelte**        | مكتبة مكونات عصرية تدعم الوضع الليلي والنهاري          |

## ✨ الميزات الأساسية للمنصة (Features)

- [x] **هيكلة تعليمية متكاملة:** تنظيم هرمي سلس (مراحل -> سنوات -> مواد -> فصول دراسية).
- [x] **تبويبات الفصول الدراسية (Segmented Tabs):**
  - عرض المواضيع مقسمة حسب الفصول الثلاثة (الفصل 1، الفصل 2، الفصل 3).
  - تصميم Segmented Control عصري ومتوسط الصفحة بأسلوب Shadcn UI.
  - تحديد تلقائي ذكي لأول فصل يحتوي على مواضيع.
- [x] **شريط فلترة سياقي (Contextual Filter Bar):**
  - فلاتر أفقية أنيقة لتصفية المواضيع حسب: النوع (فرض/اختبار/درس)، المصدر (رسمي/مقترح)، والسنة.
  - مفتاح تبديل (Toggle Switch) لعرض المواضيع مع التصحيح فقط.
  - تصميم Mobile-First: الفلاتر تتراص عمودياً على الهواتف وأفقياً على الشاشات الكبيرة.
  - ألوان متوافقة مع Shadcn UI تتكيف تلقائياً مع الوضع الليلي والنهاري.
- [x] **قارئ PDF مدمج متطور:**
  - عرض الفروض والاختبارات مباشرة في الصفحة (بدون تحميل إجباري).
  - نافذة منبثقة ملء الشاشة (Fullscreen Modal) لتجربة قراءة خالية من التشتت.
  - التنقل المباشر والسلس بين "الموضوع" و"الحل النموذجي" بنقرة واحدة.
- [x] **طباعة وتحميل ذكي:**
  - زر تحميل 📥 مباشر للمواضيع والحلول.
  - طباعة 🖨️ سلسة وسريعة تعمل في الخلفية بدون فتح النوافذ المزعجة.
- [x] **تتبع ذكي للمواضيع (Smart Tracking):**
  - تتبع المواضيع المحلولة والمقروءة عبر المتصفح (`localStorage`).
  - ظهور علامة (✓) وتغير لون الزر للمواضيع التي قام الطالب بالاطلاع عليها لتسهيل المراجعة.
- [x] **تصفح وتصميم عصري (UI/UX):**
  - متجاوب 100% مع شاشات الهواتف والحواسيب (Mobile-First).
  - دعم أصيل للغة العربية (RTL Direction).
  - وضع ليلي / نهاري (Dark & Light Mode) لراحة العين.
  - بطاقات مواد وسنوية (Glassmorphism) مع إحصائيات المواضيع المتاحة.
  - إنجازات وتحديات بصرية مبهجة مع نظام تعليقات فورية.
  - نظام مسار التنقل (Breadcrumb) لسهولة العودة والابحار في المنصة.
  - ألوان ومكونات Shadcn-Svelte لتناسق بصري كامل في كلا الوضعين.
- [x] **لوحة تحكم (Admin Panel):**
  - حماية مسار الإدارة عن طريق مُصادقة (Authentication) مُشفرة بـ `bcryptjs`.
  - إدارة شاملة (إضافة، تعديل، حذف، ومعاينة) للأطوار، السنوات، المواد، والوثائق.
  - واجهة Dashboard عصرية وداعمة للهواتف المحمولة.
- [x] **تتبع دقيق للمواضيع (Algerian Exam Codes):**
  - نظام ترميز دقيق لكل مادة (مثال: `PHY-SCI` للعوم التجريبية).
  - تقسيم الشعب في التعليم الثانوي (جذع مشترك علوم، جذع مشترك آداب، علوم تجريبية، رياضيات...).
  - القاعدة الذهبية: توحيد كود المادة إذا كان الامتحان موحداً بين الشعب.
- [x] **محرك التمارين التفاعلي (Quiz Engine):**
  - **14 نوع سؤال تفاعلي (Moodle):** اختيار من متعدد، صحيح/خطأ، ترتيب، سحب وإفلات، ربط، ملء الفراغ، إجابة قصيرة، اختيار من قائمة، حسابي متغير، إعادة ترتيب جملة، تحديد على صورة، سحب إلى صورة، مصفوفة، مقال.
  - **تجربة مستخدم محسنة:** دعم كامل للحروف العربية في الخيارات (أ، ب، ج، د)، وواجهة مراجعة إجابات مفصلة.
  - **تخزين JSONB:** تخزين مرن للبيانات في قاعدة البيانات لسهولة التوسع ودعم أنواع أسئلة معقدة.
  - **أوضاع التشغيل:** وضع "التدريب" للتعلم الفوري، ووضع "الامتحان" الرسمي مع مؤقت زمني.
  - **توزيع عشوائي:** إمكانية خلط الأسئلة وخيارات الإجابة لضمان النزاهة.
- [x] **نظام الإنجازات والتلعيب (Gamification):**
  - 8 شارات إنجاز تلقائية (Badge System).
  - نظام نقاط (XP) ولوحة صدارة (Leaderboard) عالمية.
  - صفحة إنجازات شخصية لعرض التقدم.
- [x] **نظام النشر والتحكم (Admin & Deployment):**
  - لوحة تحكم متكاملة لإدارة المحتوى والأسئلة.
  - دعم نشر سريع عبر Docker و Caddy.

## 📂 هيكل المشروع (Directory Structure)

```
sujetstore/
├── src/
│   ├── lib/
│   │   ├── components/       # مكونات Svelte المشتركة (UI Components)
│   │   └── server/db/        # إعدادات Drizzle وجداول قواعد البيانات (Schema)
│   ├── routes/               # Pages (File-based routing):
│   │   ├── admin/                      # لوحة التحكم والإدارة (محمية بكلمة مرور)
│   │   │   ├── levels/                 # إدارة المراحل الدراسية
│   │   │   ├── years/                  # إدارة السنوات الدراسية
│   │   │   ├── subjects/               # إدارة المواد الدراسية
│   │   │   └── documents/              # رفع وإدارة الوثائق (الامتحانات والفروض)
│   │   ├── [level]/                    # صفحة السنوات (مثل: /moyen)
│   │   │   └── [year]/                 # صفحة المواد (مثل: /moyen/1am)
│   │   │       └── [subject]/          # صفحة قائمة الفروض (مثل: /moyen/1am/math)
│   └── app.html & app.css   # التصميم الأساسي للموقع
├── book/                     # دليل المنصة (Typst Manual)
├── data/
│   ├── content.db            # المحتوى التعليمي للمنهاج الجزائري
│   └── users.db              # بيانات المستخدمين (للتوسعات القادمة)
└── scripts/
    └── seed-education.ts     # سكربت ملء وبناء البيانات الأولية
```

## 🌐 النشر (Deployment)

لرفع التطبيق على خادم VPS أو سيرفر خاص، يمكنك استخدام السكربت المرفق:

```bash
./deploy.sh full    # الإعداد الأولي والرفع
./deploy.sh deploy  # لنشر تحديثات النظام اللاحقة
```
